# `GeoJSON.IO for VSCode`

## What it is?

`GeoJSON.IO for VSCode` is an open-source extension for Visual Studio Code.

It simply helps you to create, preview and edit your geojson data on a map. üó∫

## Usage

1. Install this extension. üòÄ
2. Open your geojson file or an empty file (.json„ÄÅ.txt) in VSCode.
3. Press `ctrl + shift + p`, search `View On Map` command and press `Enter`. Or just right click in your file editor window and select `View On Map`.
4. Enjoy!
