## 说明

`GeoJSON.IO for VSCode` 是一个开源的 VSCode 插件, 可以让使用者在 VSCode 中便捷地创建，预览和编辑 GeoJSON 数据。

该插件启发自[geojson.io](https://geojson.io)。

## 特性

- 支持实时编辑集合和属性信息

- 支持自定义样式渲染

- 支持两种投影模式：3D 和 墨卡托投影

- 支持多底图切换
